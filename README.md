# メモ自動保存スクリプト（Python）

このスクリプトは、入力した文字をタイムスタンプ付きでファイルに追記していく簡易メモアプリです。

## 特徴
- その場でメモを入力
- 日時付きで自動保存
- `memo_log.txt`に追記していく形式

## 使い方
```bash
python memo.py
exit と入力すると終了します。


---

## 🔚 まとめ

このスクリプトは、
- `input`で文字を入力
- `datetime`で時間を取得
- `with open`でファイルに書き込み

というPythonの基本がたくさん詰まっている、いい練習になります。
